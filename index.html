Recent Projects

Slideview.js

Visit Project

Slideview is a JavaScript library (no jQuery) for responsive sliders. It doesn't have any dependencies and it's small in size. Basic features are provided out of the box with various configurable options.

The code is open source on Github.

Amtai.com

Visit Project

AMTAI is a Medical Equipment manufacturer based in Raleigh, NC, and they were in need of a simple but powerful website where they could login and edit its contents themselves.

With my help they received a modern website with an easily navigable CMS.

IC Admin

Visit Project

IC Admin is an admin panel & can power small to medium size websites. The code depends on PHP 5.4+, MySQL, and the latest CodeIgniter php framework.

Among many features, it has a beautiful mobile friendly design, and Google+ authentication.
